Turn Properties

Here is a screenshot of the turn properties window.


Main properties

Track text field defines which track this turn is linked to.
Start time text field defines the timecode of the start boundary.
End time text field defines the timecode of the end boundary.

Transcription

Speech/no speech radio button defines whether this turn is a speech turn or not. Theses radio buttons enable or disable the next controls.
Overlapping speech checkbox is active if this speech turn is overlapped with another one. This checkbox is read only.
Overlapping noise checkbox is active if there is a noise overlapping this speech turn. This checkbox is read only.
Speaker combobox defines the speaker of the speech turn.
Dictionary button opens a speakers dictionary in order to choose one of it.
Language combobox defines the language of the speech turn.
Dialect combobox defines the dialect of the speech turn.
Additional properties supplies other properties. Theses properties are specified in a configuration file.
No speech combobox defines, if the turn is not a speech turn, the type of turn.
