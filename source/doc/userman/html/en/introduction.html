TranscriberAG Manual

What is TranscriberAG ?

The TranscriberAG application enables you to create and edit annotations related to the audio track of audio or video files. You can use TranscriberAG to perform a variety of tasks related to audio data annotation:

  • signal segmentation
  • text transcription
  • rich transcription annotations
  • named entities annotation
  • ...

TranscriberAG also offers a variety of features to comfortably browse through annotations and signal and to enhance signal output (tempo / pitch  variation) in order to ease and accelerate the annotation task.

The "AG" suffix of TranscriberAG stands for "Annotation Graph", which is the data model supporting all annotations made on signal. The associated XML-based file format is named "TAG", and file names are suffixed ".tag".

For more information on annotations graphs see Steven Bird's publications at LDC.
