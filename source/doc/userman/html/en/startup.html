Getting Started



To Start TranscriberAG

You can start TranscriberAG in the following ways:

Desktop QuickLaunch Icon (Windows only)

Double-click on the TranscriberAG icon

Command line: execute the following command:
TranscriberAG
Displaying command line options:
TranscriberAG --help

When You Start TranscriberAG


The first time you start TranscriberAG, you will be prompted to define an identifier. This identifier will be afterwards used to tag annotations files you create or edit.


This happens only once, next time you'll start TranscriberAG you'll go directly to Main window
