Getting AGLIB

AGLIB sources and binaries can be downloaded from the SourceForge AGTK download site, in a section named AGLIB.

Source packages are released in two forms: tar-gzipped package and RPM package.

Binary packages are currently released for two platforms: Linux and Mac OS X.

The latest copy of AGLIB source can be obtained from the SourceForge AGTK CVS repository. Use AGLIB for the module name. Please note that files in the CVS may be under development, and may change at any time.
